# Генератор на баркодове

Професионално уеб приложение за генериране на баркодове, подходящо за търговия и продукти. Работи изцяло в браузъра — не е необходим сървър.

## Функции

- **4 формата на баркодове:** EAN-13, Code 128, QR Code, UPC-A
- **Преглед в реално време** — баркодът се обновява докато пишете
- **Валидиране на въведените данни** — брой цифри, контролни цифри, допустими знаци
- **Експорт в PDF** — баркодът се центрира на A4 страница с данните и незадължителен продуктов етикет
- **Адаптивен дизайн** — работи на телефони и таблети
- **Без инсталация** — чист HTML/CSS/JS с CDN библиотеки

## Как да стартирате локално

1. Клонирайте или изтеглете хранилището:
   ```bash
   git clone https://github.com/your-username/barcode-generator.git
   cd barcode-generator
   ```

2. Отворете `index.html` в произволен съвременен браузър — без стъпка за изграждане, без нужда от сървър.

   > **Съвет:** Ако получавате CDN грешки без интернет връзка, използвайте локален сървър:
   > ```bash
   > npx serve .
   > # или
   > python -m http.server 8080
   > ```

## Как се използва

1. **Изберете тип баркод** от четирите карти.
2. **Въведете данните** в появилото се поле:
   - **EAN-13** — точно 13 цифри (контролната цифра се проверява автоматично)
   - **Code 128** — произволен ASCII текст, 1–80 знака
   - **QR Code** — произволен текст или URL до ~2 900 знака
   - **UPC-A** — точно 12 цифри (контролната цифра се проверява автоматично)
3. Наблюдавайте **прегледа на баркода в реално време** докато пишете.
4. Натиснете **Напред → Експорт** когато баркодът изглежда правилно.
5. По желание добавете **продуктов етикет** (напр. „Премиум кафе на зърна 250г").
6. Натиснете **Изтегли PDF** за да запазите A4 PDF файл с центриран баркод.

```

├── css/

```
